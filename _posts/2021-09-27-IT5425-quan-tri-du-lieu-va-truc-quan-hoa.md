## IT5425 Quản trị dữ liệu và trực quan hóa (Data management and visualization)

-	Khối lượng (Credits): 2(2-1-0-4)
-	Học phần tiên quyết (Prerequisite): Không (None)
-	Học phần học trước (Pre-courses): Không (None)
-	Học phần song hành (Corequisite Courses): Không (None)

Mục tiêu: Môn học này nhằm cung cấp những kiến thức nền tảng về 2 vấn đề quản trị và trực quan hoá dữ liệu. Môn học dẫn dắt sinh viên từ bài toán tổ chức dữ liệu tới làm sạch và tiền xử lý dữ liệu, sử dụng các công cụ trực quan hoá để phân tích thăm dò dữ liệu và cuối cùng là trình diễn, kể chuyện với dữ liệu. Sinh viên thực hành trên ngôn ngữ Python, trực tiếp trên tập dữ liệu của các bài toán thực tiễn. Kết thúc môn học, sinh viên có thể lựa chọn các giải pháp và công cụ lưu trữ và trực quan hoá dữ liệu trong bài toán thực tế. 

Objectives: This course provides basic foundations on data management and visualization. Students are trained to design and propose solutions to store, manage, integrate data and finally able to visually present a story on the data and data insights. Real-life applications and datasets will be provided for practicing on Python language and libraries throughout the lessons. After finishing the course, students will know how to represent and analyze data with visual tools, as well as know how to apply each type of chart to different purposes depending on the characteristics of the data.

## Nội dung giảng dạy / Course content

### Chương 1. Tổng quan về trực quan hoá dữ liệu  
*Chapter 1: Overview of data visualization*

- 1.1. Giới thiệu / Introduction  
- 1.2. Tại sao cần trực quan hoá dữ liệu / Why data visualization  
- 1.3. Vai trò của trực quan hoá dữ liệu / Role of data visualization  
- 1.4. Các loại hình trực quan hoá dữ liệu / Types of data visualization  

### Chương 2. Mô hình và mã hoá trực quan  
*Chapter 2: Visual models and encoding*

- 2.1. Tính chất của dữ liệu và các loại thang đo dữ liệu / Properties of data and measurement levels  
- 2.2. Phân loại đối tượng biểu diễn / Visual marks  
- 2.3. Phân loại biến thị giác / Visual channels  
- 2.4. Mã hoá hình ảnh / Visual encoding  

### Chương 3. Nhận thức trực quan bằng đồ hoạ  
*Chapter 3: Graphical perception*

- 3.1. Phát hiện tín hiệu / Signal detection  
- 3.2. Ước lượng độ lớn / Magnitude estimation  
- 3.3. Xử lý tiền chú ý / Pre-attentive processing  
- 3.4. Sử dụng nhiều mã hoá trực quan / Using multiple visual encodings  
- 3.5. Luật gom nhóm Gestalt / Gestalt grouping principles  

### Chương 4. Trực quan hoá dữ liệu đa chiều  
*Chapter 4: Visualization for multi-dimensional data*

- 4.1. Các loại hệ trục toạ độ / Coordinate systems and axes  
- 4.2. Trực quan hoá các giá trị / Visualizing amounts  
- 4.3. Trực quan hoá phân phối / Visualizing distributions  
- 4.4. Trực quan hoá tỉ lệ / Visualizing proportions  
- 4.5. Trực quan hoá sự tương quan / Visualizing relationships  
- 4.6. Trực quan hoá xu hướng / Visualizing trends  
- 4.7. Trực quan hoá sự không chắc chắn / Visualizing uncertainty  

### Chương 5. Trực quan hoá dữ liệu đồ thị  
*Chapter 5: Visualization for graphs*

- 5.1. Các tính chất của dữ liệu đồ thị / Graph data properties  
- 5.2. Trực quan hoá dữ liệu đồ thị / Mechanisms for graph visualization  

### Chương 6. Nguyên tắc thiết kế hình vẽ  
*Chapter 6: Principles of figure design*

- 6.1. Nguyên tắc tỷ lệ mực in / The principle of proportional ink  
- 6.2. Nguyên tắc xử lý điểm chồng lấn / Handling overlapping points  
- 6.3. Các vấn đề về sử dụng màu sắc / Issues in the use of color  
- 6.4. Chia khung hình vẽ / Multi-panel figures  
- 6.5. Tiêu đề, chú thích và bảng biểu / Titles, captions, and tables  
- 6.6. Cân bằng dữ liệu và bối cảnh / Balancing data and context  
- 6.7. Vấn đề với biểu đồ 3D / Problems with 3D charts  

### Chương 7. Trực quan hoá bản đồ  
*Chapter 7: Map visualization*

- 7.1. Cơ sở lý thuyết / Theoretical foundations  
- 7.2. Các loại trực quan hoá dữ liệu bản đồ / Types of map data visualization  

### Chương 8. Trực quan hoá tương tác  
*Chapter 8: Interactive visualization*

- 8.1. Giới thiệu / Introduction  
- 8.2. Nguyên tắc thiết kế / Design principles  
- 8.3. Kỹ thuật tương tác: lọc, thu phóng, chọn, chuyển đổi chế độ hiển thị / Interaction techniques: filtering, zooming, selection, view transformation  
- 8.4. Trực quan hoá động / Animated visualization  
- 8.5. Công cụ và thư viện / Tools and libraries  
- 8.6. Ứng dụng thực tiễn / Practical applications  

### Chương 9. Kể chuyện bằng dữ liệu  
*Chapter 9: Storytelling with data*

- 9.1. Giới thiệu / Introduction  
- 9.2. Nguyên tắc cơ bản của kể chuyện bằng dữ liệu / Fundamental principles of data storytelling  
- 9.3. Kỹ thuật trực quan hoá cho kể chuyện / Visualization techniques for storytelling

# Hướng dẫn bài tập lớn

Xây dựng **dashboard** cho một nguồn dữ liệu cụ thể.

- **Công cụ:** Power BI, Apache Superset, Tableau, Python (Plotly, Dash, Streamlit), hoặc công cụ tương đương.
- **Nguồn dữ liệu:** Dataset phải tìm được trên [Google Dataset Search](https://datasetsearch.research.google.com/).

## Deliverables

| Thành phần | Mô tả |
|------------|-------|
| **Mã nguồn** | Source code dashboard (hoặc file project nếu dùng Power BI, Tableau) |
| **Slide trình bày** | Slide thuyết trình demo dashboard |
| **Báo cáo** | Báo cáo phân tích rõ các kỹ thuật và guideline trực quan hoá đã học, đã áp dụng trong dashboard |

## Guideline viết phần "Áp dụng kỹ thuật" trong báo cáo

Báo cáo cần có phần phân tích **rõ ràng** các kỹ thuật và nguyên tắc trực quan hoá đã học qua các chương, và **cách áp dụng** vào từng biểu đồ/trang trong dashboard. Sinh viên trình bày theo cấu trúc dưới đây — mỗi chương có liên quan thì viết một mục tương ứng.

### Mẫu cấu trúc cho mỗi chương

```text
### Chương X: [Tên chương]

#### Kỹ thuật / Nguyên tắc đã áp dụng
- Liệt kê các mục (X.1, X.2, ...) từ chương có liên quan đến dashboard của bạn.

#### Cách áp dụng trong dashboard
- Mô tả cụ thể: biểu đồ/trang nào, dùng kỹ thuật gì, vì sao chọn cách đó.
- Có thể kèm ảnh chụp màn hình minh hoạ.

#### Lưu ý / Điều chỉnh (nếu có)
- Nếu có thay đổi so với lý thuyết, giải thích lý do.
```

### Gợi ý nội dung theo từng chương

| Chương | Nội dung cần trình bày trong báo cáo |
|--------|-------------------------------------|
| **Chương 1** | Lý do chọn dataset; vai trò của trực quan hoá trong bài toán; các loại hình trực quan đã dùng (exploratory, explanatory, …). |
| **Chương 2** | Phân loại dữ liệu (nominal, ordinal, quantitative); visual marks (points, lines, areas) và visual channels (position, size, color, …) đã dùng; lý do chọn encoding cho từng biến. |
| **Chương 3** | Cách tận dụng pre-attentive processing; Gestalt principles (proximity, similarity, …) trong layout; ước lượng độ lớn (length vs area vs angle) — biểu đồ nào dùng, có phù hợp không. |
| **Chương 4** | Loại biểu đồ cho amounts, distributions, proportions, relationships, trends; hệ trục toạ độ; có trực quan hoá uncertainty không. |
| **Chương 5** | *(Nếu có dữ liệu đồ thị)* Tính chất dữ liệu đồ thị; cơ chế layout (force-directed, hierarchical, …); cách biểu diễn node/edge. |
| **Chương 6** | Nguyên tắc proportional ink; xử lý overlapping; dùng màu sắc (palette, accessibility); multi-panel; tiêu đề, chú thích; cân bằng data–context; tránh biểu đồ 3D không cần thiết. |
| **Chương 7** | *(Nếu có map)* Loại map visualization; cơ sở lý thuyết (projection, geocoding); cách mã hoá dữ liệu trên bản đồ. |
| **Chương 8** | Kỹ thuật tương tác: lọc, zoom, chọn, chuyển view; có animation không; công cụ/thư viện dùng. |
| **Chương 9** | Cấu trúc câu chuyện; nguyên tắc storytelling; phong cách tường thuật; tương tác và khám phá trong kể chuyện. |

**Lưu ý:** Không bắt buộc phải có nội dung cho mọi chương. Chỉ viết các chương **có liên quan** đến dashboard của bạn; chương không áp dụng có thể bỏ qua hoặc nêu ngắn gọn lý do.
